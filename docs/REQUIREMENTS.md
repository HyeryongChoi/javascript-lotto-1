## 🎯 기능 목록

### View

- [ ]  입력받는 객체 생성하기
	- [x]  메서드) 구입 금액 입력받기
    - [ ]  예외) 숫자와 공백 받기
    - [ ]  예외) 1000원 단위만 받기 (몫, 나머지 체크)
    - [ ]  예외) 0으로 시작하는 경우 예외 처리
  - [x]  메서드) 당첨 번호 입력받기
    - [ ]  예외) 숫자와 ,와 공백 입력받기
    - [ ]  예외) 1 ~ 45 사이의 숫자 입력받기
    - [ ]  예외) 6개 입력 받기
    - [ ]  예외) 중복되지 않는 숫자 입력받기
  - [x]  메서드) 보너스 번호 입력받기
    - [ ]  예외) 1 ~ 45 사이의 숫자와 공백 입력받기
    - [ ]  예외) 1개 입력 받기
    - [ ]  예외) 당첨 번호와 중복되지 않는 숫자 입력받기
  - [ ]  메서드) 재시작/종료 여부를 입력받기
    - [ ]  예외) y 혹은 n 입력받기

- [ ]  출력하는 객체 생성하기
	- [x]  메서드) 로또 구입 결과 출력하기
  - [ ]  메서드) 당첨 통계 출력하기
